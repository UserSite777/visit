🏰 Подземелья и сокровища

Инновационная текстовая RPG, разработанная с использованием современных технологий Android — Jetpack Compose и Kotlin. Игра демонстрирует актуальные подходы мобильной разработки 2025 года.

🚀 Современные технологии разработки

⚡ Jetpack Compose — будущее Android UI

С 2025 года Jetpack Compose стал стандартом индустрии. Более 125,000 приложений в Play Store используют эту технологию. Наша игра полностью построена на декларативном подходе к созданию пользовательских интерфейсов.

📱 Архитектура WebView + Compose

Игра использует гибридный подход: HTML5 контент отображается через WebView внутри Compose UI, обеспечивая максимальную производительность и кроссплатформенность.

🔗 JavaScript Bridge

Обмен данными между нативным Kotlin кодом и JavaScript реализован через современный WebView API, позволяя сохранять игровой прогресс и синхронизировать состояние.

🎨 Material Design 3

Интерфейс следует актуальным принципам Material You, адаптируясь под системные настройки пользователя и обеспечивая современный UX.

⚙️ Оптимизированная производительность

Используются последние оптимизации Compose 2025.05.01: Pausable Composition, фоновая предзагрузка текста и улучшения LazyLayout.

🛠️ Техническая реализация

📦 Структура проекта

MainActivity — основная Activity с Compose UI
/assets/ — HTML, CSS, JavaScript файлы игры
GameViewModel — управление состоянием
WebViewBridge — связь с JavaScript

💾 Локальное хранение

Игровой прогресс сохраняется в SharedPreferences и локальной базе данных Room, обеспечивая быстрый доступ и надёжность данных.

🌐 Офлайн режим

Все ресурсы игры упакованы в APK, что позволяет полноценно играть без подключения к интернету. Perfect для мобильных устройств!

🔄 Реактивность

Используется StateFlow и Compose State для реактивного обновления UI при изменениях игрового состояния.
